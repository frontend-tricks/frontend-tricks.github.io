Scroll To Top

Writing the Markup

                        <a href="#" class="btn-top">top</a>
                    </body>

Writing the Css

                    .btn-top {
                        width: 40px;
                        height: 40px;
                        position: fixed;
                        bottom: 50px;
                        right: 100px;
                        display: none;
                        text-indent: -9999px;
                        background: #000 url('ico-top.png') no-repeat;
                    }

Writing the jQuery

                    $(document).ready(function () {

                        $(window).scroll(function () {
                            if ($(this).scrollTop() > 100) {
                                $('.btn-top').fadeIn();
                            } else {
                                $('.btn-top').fadeOut();
                            }
                        });

                        $('.btn-top').click(function () {
                            $("html, body").animate({
                                scrollTop: 0
                            }, 600);
                            return false;
                        });

                    });
