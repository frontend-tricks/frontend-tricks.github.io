Writing the Css

                ul {
                    font: 14px Verdana, Geneva, sans-serif;
                    text-align: justify;

                    /* Обнуляем для родителя*/
                    line-height: 0;
                    font-size: 1px; /* 1px для Opera */

                    /* Лекарство для IE6-7*/
                    text-justify: newspaper;
                    zoom:1;

                    /* Включаем в работу последнюю строку*/
                    text-align-last: justify;
                }

                ul:after {
                    width: 100%;
                    height: 0px;
                    visibility: hidden;
                    overflow: hidden;
                    content: '';
                    display: inline-block;
                }

                ul li {
                    display: inline-block;
                    line-height: normal;
                    vertical-align: top;
                    font-size: 14px;
                }
